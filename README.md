## 项目简介

这是一个专为 EvaHan2026 评测任务中的 Task B 版面分析数据修复设计的 Web 标注工具。支持对古籍图像进行文本、图像、印章、书边等四类区域的标注和修正。

## 项目结构

```
workspace/
├── app.py                 # 后端服务代码
├── index.html             # 前端页面
├── demo.json              # 官方原始数据切片
├── update.json            # 中间标注数据存储文件（任务开始前需确保为空）
├── Dataset_B/             # 官方图像文件夹
│   ├── b_1122.jpg
│   ├── b_1123.jpg
│   └── ...
└── README.md              # 本文档
```

### 文件说明

- **app.py**: FastAPI 后端服务，负责数据加载、保存、导出等 API 接口
- **index.html**: 前端界面，提供图像展示、标注绘制、数据管理等功能
- **demo.json**: 官方提供的原始标注数据切片，用于实际标注
- **update.json**: 临时存储当前标注进度，包含原始标注 + 新增/修改标注
- **Dataset_B/**: 存放待标注的古籍图像文件

## 使用说明

### 1. 任务初始化

1. 修改需要标注的 json文件路径
2. **确保 update.json 为空或不存在**
3. 确认 Dataset_B 文件夹包含对应的图像文件

### 2. 启动服务

```bash
python app.py
```

服务启动后，访问：http://localhost:7896

### 3. 标注操作

#### 绘制模式

- **拖拽模式**：按住鼠标左键拖动绘制矩形框
- **点击模式**：点击对角线两点确定矩形框

#### 标注类型


| 类型             | 颜色 | 说明         |
| ---------------- | ---- | ------------ |
| 文本 (text)      | 红色 | 标注文字区域 |
| 图像 (image)     | 绿色 | 标注图片插图 |
| 印章 (seal)      | 蓝色 | 标注印章     |
| 书边 (book_edge) | 橙色 | 标注书籍边缘 |


#### 快捷键

- `A` / 鼠标滚轮向上：上一张图片
- `D` / 鼠标滚轮向下：下一张图片
- `右键点击区域`：选中区域
- `右键再次点击已选区域`：删除选中的**新增区域**
- `空格键`：删除当前选中的新增区域
- `选中区域 + 点击标注类型按钮`：修改区域标签

#### 区域管理

- **原始标注**（虚线框）：来自官方数据，不可删除，仅可修改类型
- **新增标注**（实线框）：用户新绘制的区域，可删除和修改

### 4. 保存与导出

#### 自动保存

每次新增或修改标注后，数据会自动保存到 update.json

#### 导出数据

1. 点击右下角 **"💾 保存并导出"** 按钮
2. 输入导出文件名（如 `task_1_completed`）
3. 系统将：
   - 导出完整标注数据为 `task_1_completed.json`
   - 清空 update.json
   - 自动关闭服务器


## 数据格式

```json

```

## 注意事项

1. **任务开始前**：务必确保 update.json 为空
2. **图像路径**：Dataset_B 文件夹中的图像文件名必须与 JSON 中的路径对应
3. **数据备份**：定期备份 update.json，防止意外丢失
4. **原始标注**：带虚线的原始标注不可删除，仅能修改类型
5. **导出后处理**：导出操作会清空 update.json 并关闭服务器，请确认完成后再执行

## 常见问题

### Q: 如何重新开始标注任务？

A: 删除或清空 update.json，重启服务即可从原始数据重新开始。

### Q: 误删区域如何恢复？

A: 中间数据已保存在 update.json，可从备份中恢复。建议定期备份。

### Q: 为什么原始标注无法删除？

A: 原始标注来自官方数据，设计上仅允许修改类型，不允许删除，以保证数据完整性。

### Q: 如何修改已标注区域的类型？

A: 右键点击区域选中后，点击右侧对应的标注类型按钮即可修改。
